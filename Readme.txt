How to run the Attendance Monitoring System (AMS) using PHP and MySQL 
1. Download the  zip file
2. Extract the file and copy amsfolder
3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)
4. Open PHPMyAdmin (http://localhost/phpmyadmin)
5. Create a database with name ams
6. Import ams.sql file(given inside the zip package in SQL file folder)
7.Run the script http://localhost/ams (frontend)
